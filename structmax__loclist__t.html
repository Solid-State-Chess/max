#include <piecelist.h>

Detailed Description

A piece list storing the locations of one piece type indexed by max_lidx_t.

This typedef has a variable size, and is thus only operated on by pointers. In practice, it is provided so that piece list functions have a uniform interface for piece lists of different static sizes.
