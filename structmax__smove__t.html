#include <move.h>

Detailed Description

A move with from and to square, plus flags stored in a separate byte.

This structure uses three bytes to represent a move with positions already stored in 0x88 form occupying one byte each - if memory is not a top priority this representation enables faster processing to avoid packing and unpacking destination squares.
