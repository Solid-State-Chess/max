#include <state.h>

Detailed Description

A struture representing data that must be tracked separately from the actual chess board.

In the #max_chessboard_t, this structure is not used directly, rather it is stored in a max_state_stack_t.

◆ check

max_check_t max_state_t::check[2]

An array of two separate check structures, if one or both of these structures is valid then the side to play is in check and must escape in order to continue the game.

The first (index 0) check structure will always be set to indicate single check, while the second will be filled if the king is in double check.
