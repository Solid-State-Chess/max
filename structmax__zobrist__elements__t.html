Chessboard » Zobrist Keys

#include <zobrist.h>

Detailed Description

Static arrays used to incrementally compute zobrist hash keys.

These contribute the largest memory footprint to the board by itself, but allow for huge performance gains in engine processing and threefold repetition.

An array indexed by the side that a piece is on, the piece type, and the position of the piece.

Zobrist hashes are internally built by XORing the current hash with this whenever a piece is added to or removed from the board
