max_engine_t Struct Reference
Chess Engine

#include <engine.h>

Detailed Description

All state required by the chess engine to search a game tree and evaluate positions.

Includes a pointer to a max_board_t that will be used to search a tree by making and unmaking moves, and configurable evaluation parameters.
