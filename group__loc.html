Board Location Indices
Chessboard

Detailed Description

Rank and file data stuffed into one byte using both the 0x88 and 6 bit LSF encoding schemes.

0x88 is primarily used by the board for its index difference properties, but directly indexing arrays using this encoding wastes 64 bytes of space, so for lookup table flexibility both encoding schemes are defined.
